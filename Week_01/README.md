#### 1. 五毒神掌
1. 5-10 分钟读题和思考，如果没有思路，看题解，默写代码
2. 马上自己写，提交LeetCode，多种解法，体会优化
3. 24 小时之后，再重复做题
4. 一周后重复做题
5. 面试前一周恢复性训练


#### 2. 切题四件套
1. 理清题意，确定题目的要求
2. 想尽可能多的解法，对比几种写法的时空复杂度，找到比较好的解法
3. 尽可能多地动手写
4. 测试用例

#### 3. 思路: 找最近重复子问题

#### 5. 做的不好的地方，需要改正
1. 读题不够深入，不能理解题目意思
2. 分不清轻重，花了太多时间在每一日题上，忽视了课程中的题
3. 看了题解有时间也半天理解不了，需要更专心
4. 下周以课程中题为主，可以先记下每日一题，有时间再刷，课程为主！！！
